facebookConnect()method 
public static function facebookConnect(callback:Function):void

Connects to Facebook if possible and tells the callback if it did so. The game must be run from within the CastleKingdom facebook app page in order to recieve the benefits of the Facebook API. The user must also select allow on the pop-up that ensues. Ensure that your browser is not supressing this pop-up.

Parameters

callback:Function — A callback function as a parameter, this function must have the following signature callback(ready:Boolean):void

facebookFriends()method 
public static function facebookFriends(callback:Function, justNames:Boolean = false):void

facebookConnect must have been called before this method is called. Provides the callback with a list of friends which is a simple object with an id and name field. If justNames is true, then the array contains only the string names of each friend.

Parameters

callback:Function — A callback function of the following form callback(friends:Array, fail:Object)
 
justNames:Boolean (default = false) — Whether the array should contain just names or not
