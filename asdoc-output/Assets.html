Classpublic class Assets
InheritanceAssets Object

This class keeps track of different dictionaries filled with assets for Castle Kingdom (attack units, defense units, castle upgrades) that pertain to different skins. The overall dictionary is a map from String (skin name) to a dictionary for the assets for that skin. The containing dictionaries are maps from Strings to objects.
